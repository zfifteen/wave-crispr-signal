# Tools

Utility scripts for demos, preprocessing, postprocessing, and support tasks.

## Tool Overview

- `demo_dna_storage_hypothesis.py` - DNA storage hypothesis demonstration
- `demo_invariant_features.py` - invariant feature demonstration
- `demo_topological_analysis.py` - topological analysis demonstration
- `pain_management_demo.py` - domain demo script
- `quick_validation_demo.py` - quick validation demonstration
- `check_repo_policy.py` - deprecated non-blocking compatibility command

## Usage

```bash
python tools/demo_dna_storage_hypothesis.py
python tools/quick_validation_demo.py
python tools/check_repo_policy.py
```

## Notes

`check_repo_policy.py` no longer enforces repository layout and always exits successfully during reset.
