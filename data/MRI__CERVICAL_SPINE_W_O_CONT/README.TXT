VA DICOM images and imaging test report
ReadMe

This .zip file includes diagnostic quality DICOM images and a copy of your imaging test report.

Here are a few tips for using this .zip file:

1. You can use the .zip file to share your medical images and report with another provider.  

2. All of your diagnostic quality images are located in the DICOM folder and have the file 
   extension *.dcm.  

3. Providers use special software to view these DICOM images, so you may not be able to view
   them on your device.

4. The DICOMDIR file is a directory file that other computer systems can use to document the 
   DICOM files that are included.

5. The Report.txt file is a copy of your imaging test report.
